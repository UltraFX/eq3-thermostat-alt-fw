STM8L15x/16x/05x/AL31-L Standard Peripherals Library Template update History

1.6.1 / 30-September-2014

Main Changes

  • STM8L15x_StdPeriph_Template: Template projects
    • ST Visual Develop (STVD) software toolchain
      • STVD Version 4.3.6
      • Cosmic STM8 32K compiler Version 4.3.12 (or later)
      • Raisonance STM8/ST7 C compiler Version 2.52.13.0324
    • Raisonance IDE RIDE7 (RIDE) software toolchain
      • Version: RIDE7 IDE:7.48.13.0324, RKit-STM8 for 2.52.13.0324
    • IAR Embedded Workbench for STM8 IDE (EWSTM8) software toolchain
      • Version v1.42.1

V1.6.0 / 28-June-2013

Main Changes

  • Add the support of the STM8L05x Value line devices in STVD and EWSTM8 and RID7 project templates
    • Update STVD workspaces for both Cosmic and Raisonance compilers and EWSTM8 templates as follow:
      •  Rename stm8l1526-eval-ld project to STM8L15X_LD(STM8L15xx2_3) for the STM8L15x Low density devices.
      •  Rename stm8l1526-eval project to STM8L15X_MD(STM8L15xx4_6) for the STM8L15x Medium density devices.
      •  Rename stm8l1528-eval project to STM8L15X_HD(STM8L15xx8_16xx8) for the STM8L15x High density devices.
      • Add STM8L05X_LD_VL(STM8L05xx3) project for the STM8L05x Low density devices.
      • Add STM8L05X_MD_VL(STM8L05xx6) project for the STM8L05x Medium density devices.
      • Add STM8L05X_HD_VL(STM8L05xx8) project for the STM8L05x High density devices.
    • Update RIDE7 template project as follow
      • Rename stm8l1526-eval workspace to STM8L15xx4_6 for the STM8L15x Medium density devices.
      • Rename stm8l1528-eval workspace to STM8L15xx8_16xx8 for the STM8L15x High density devices.
      • Add STM8L15xx2_3 workspace for the STM8L15x Low density devices.
      • Add STM8L05xx3 workspace for the STM8L05x Low density devices.
      • Add STM8L05xx6 workspace for the STM8L05x Medium density devices.
      • Add STM8L05xx8 workspace for the STM8L05x High density devices.
      Known limitations
  • The available versions of  STVD, EWSTM8 and RID7 IDEs don't support yet the STM8AL products line devices.

V1.5.0 / 13-May-2011

Main Changes

  • Add the support of the STM8L15x Low density devices in STVD and EWSTM8 project templates
    • Update STVD workspaces for both Cosmic and Raisonance compilers by adding 'stm8l1526-eval-ld' project for the STM8L15x Low density devices
    • Update EWSTM8 template project by adding 'stm8l1526-eval-ld' workspace for the STM8L15x Low density devices.    
      Known limitations
  • The current available version of  RIDE7 IDE doesn't support yet the STM8L15x Low density devices.
